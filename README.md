----------- Solar Panel Anomaly Detection -----------------

---- Description ----
Deep learning project (USTH, 2025) for classifying thermal images of solar panels.  
Implemented CNN baseline and Vim-Tiny model using PyTorch on Google Colab with GPU.

---- How to Run ----
- Upload the dataset to your Google Drive.  
- Open the notebook in Google Colab.  
- Run the cells in order (GPU recommended).  

---- Results ----
- Accuracy ~27% on test set.  
- Best detection: MultiHotSpot & SingleByPassed.
